# Project Title

AMAZON SCRAPING TASK

## Description

A simple web-scraping task.
Backend - ExpressJS
Frontend - HTML/CSS/Vanila JavaScript

## Getting Started
clone the respository

### Dependencies

* Install dependencies
```
npm install
```

### Executing program
#### Backend (ExpressJS)
* To run in Development mode
```
npm run-dev
```

* To run in Production mode
```
npm start
```

#### Frontend 
* frontend/index.html
